# Zona Meiga Tours - prototipo estático placeholder

Proyecto estático para GitHub Pages.

## Estado

- Contenido, imágenes, opiniones, colaboradores, teléfono y email son placeholder/dummy.
- La estructura está alineada con la arquitectura prevista: Inicio, Actividades, Servicios, Blog, Galería, Sobre nosotros y Contacto.
- Los formularios, calendario, pagos online, newsletter, reservas y WhatsApp están preparados visualmente, pero no tienen funcionalidad real.

## Probar en local

```bash
python -m http.server 8080
```

Abrir `http://localhost:8080`.
